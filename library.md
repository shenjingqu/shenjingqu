  <h1>📚 GitHub 注册教程</h1>
  <p>要在这个贴吧留言，需要先有一个 GitHub 账号。跟着下面的教程，5分钟就能搞定！</p>
  
  <hr>
  
  <h2>📝 第一步：打开 GitHub 官网</h2>
  <p>在浏览器打开：<a href="https://github.com" target="_blank" rel="noopener noreferrer">https://github.com</a></p>
  
  <h2>📝 第二步：点击注册按钮</h2>
  <p>点击页面右上角的 <strong>Sign up</strong> 按钮</p>
  
  <h2>📝 第三步：填写信息</h2>
  <ul>
    <li><strong>Email</strong>：填写你的邮箱（QQ邮箱、163邮箱都可以）</li>
    <li><strong>Password</strong>：设置一个密码（至少8位，包含字母和数字）</li>
    <li><strong>Username</strong>：设置一个用户名（英文+数字，比如你的名字缩写）</li>
  </ul>
  
  <h2>📝 第四步：验证邮箱</h2>
  <p>GitHub 会发送一封验证邮件到你填写的邮箱，去邮箱里点击验证链接即可。</p>
  
  <h2>📝 第五步：完成注册</h2>
  <p>验证成功后，你的 GitHub 账号就注册好了！</p>
  
  <hr>
  
  <h2>🎉 注册完成后</h2>
  <p>返回 <a href="/">正宗菌吐槽贴吧</a>，登录你的 GitHub 账号，就可以在下面留言反馈了！</p>
  
  <div style="background: #e8f8f5; padding: 15px; border-radius: 12px; margin-top: 20px;">
    <p>💡 <strong>小提示</strong>：GitHub 账号是免费的，注册很简单。如果有任何问题，可以问身边的同学或上网搜索教程。</p>
